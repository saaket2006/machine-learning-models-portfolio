# Machine Learning Models Portfolio

This repository contains multiple machine learning projects covering:

- Regression
- Classification
- Clustering
- Support Vector Machine
- Neural Networks

## Projects Included:

1. **House Price Prediction** (Linear Regression)
2. **Iris Flower Classification** (KNN, Decision Tree)
3. **Customer Churn Prediction** (Logistic Regression, Random Forest)
4. **Stock Market Clustering** (K-Means)
5. **Sentiment Analysis** (SVM, Neural Networks)

## Technologies Used:
- Python
- Pandas
- Scikit-Learn
- TensorFlow
- Matplotlib
- Seaborn
